# User Management System

A simple Flask-based User Management System built using Python, Flask, SQLite, HTML, and CSS.

## Features
- Add users
- Store user data in SQLite database
- Display users in a table
- Simple and clean UI
- Flask backend integration

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Run Locally

1. Install Flask

pip install flask

2. Run the app

python app.py

3. Open browser

http://127.0.0.1:5000
